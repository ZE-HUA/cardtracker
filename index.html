用戶識別碼
每個人設定不同的識別碼，雲端資料就完全隔離。只要識別碼相同，同一人的多台裝置就能同步。
☁️ 跨裝置同步（Cloudflare KV）
設定後換裝置也能保留所有卡片
第一步：建立 KV 空間（一次性）
1. Cloudflare 儀表板 → Workers & Pages → KV
2. 點 Create a namespace，名稱輸入 SNKR_DB

第二步：綁定到 Worker
3. 前往你的 Worker → Settings → Variables
4. KV Namespace Bindings → Add binding
Variable name: DB / Namespace: SNKR_DB
5. 點 Save

第三步：更新 Worker 程式碼
6. 把新版 worker.js 貼上編輯器 → Save and deploy

完成後，新增/刪除卡片會自動同步到雲端 ☁️
換裝置時點 Header 的 ☁️ 圖示即可載入所有資料。
