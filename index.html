方案 A：Cloudflare Workers（推薦）
☁️ 免費、不需電腦開機
1. 開啟 dash.cloudflare.com，登入或免費註冊
2. 左側選 Workers & Pages → Create → Create Worker
3. 把 worker.js 的內容全部貼入編輯器
4. 點 Save and deploy
5. 複製顯示的網址，例如：
  https://snkrdunk-proxy.你的名字.workers.dev
6. 貼到上方「自訂伺服器 URL」欄位 → 儲存

免費額度：每天 100,000 次請求，個人使用完全足夠。
方案 B：ngrok（電腦需開機）
🚇 臨時外部連線，設定簡單
1. 先在電腦啟動本地代理：python proxy.py
2. 下載 ngrok（ngrok.com/download）並登入
3. 執行：ngrok http 8765
4. 複製顯示的 https 網址，例如：
  https://abc-123.ngrok-free.app
5. 在上方「自訂伺服器 URL」填入（需加 /api）：
  https://abc-123.ngrok-free.app/api

⚠️ 注意：ngrok 網址每次重啟都會變更，須重新設定。
